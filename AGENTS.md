# AGENTS.md

## Project Overview

This is a static portfolio website built with HTML, CSS, and JavaScript. This site will be hosted on GitHub Pages.

## Tech Stack

- HTML5
- CSS3
- JavaScript (ES6+)
- No frameworks unless explicitly requested

## Directory Structure

assets/
css/
js/
images/
icons/
fonts/

## Coding Guidelines

- Use semantic HTML.
- Write modular, readable CSS.
- Keep JavaScript in separate files.
- Use relative paths for all assets.
- Do not use inline CSS or inline JavaScript unless necessary.
- Ensure responisve layouts for desktop, tablet, and mobile.
- Prioritize accessibility (alt text, labels, keyboard navigation).
- Keep code simple and maintainable.

## Permissions

The agent may:

- Create, modify, and delete project files as requested.
- Refactor code.
- Explain changes.

The agent must not:

- Execute Git commands.
- Install packages without permission.
- Add new dependencies without approval.
- Change project configuation unless requested.

## Output Expectations

- Explain significant architectural decisions.
- Preserve existing functionality when making changes.
- Avoid introducing unnecessary dependencies.
